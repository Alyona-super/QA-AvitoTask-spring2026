###### 1
Название: Фильтрация объявлений по диапазону цен не выполняется

Короткое описание: При выставлении в фильтрации ценового диапазона отображаются объявления, находящиеся за пределом установленного ограничения.

шаги воспроизведения:

    1. Открыть /

    2. Ввести в поле "Цена от" значение 500
    
    3. Ввести в поле "Цена до" значение 1000,
    
    4. Получить список объявлений,

    
ожидаемый результат: цена у всех отображенных объявлений находится в диапазоне от 500 до 1000 включительно.

фактический результат: отображаются объявления с ценой, находящейся за пределами установленных ограничений.

Проект: https://cerulean-praline-8e5aa6.netlify.app/,

Платформа: ПК, Chrome

Зона, в которой обнаружен баг: Главная страница, раздел фильтрации

Серьёзность: критическая,

Приоритет: высокий,

Статус: Создан.
